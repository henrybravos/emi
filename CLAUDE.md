## Project Overview

Baby shower invitation landing page with:
- Light blue (celeste) color scheme based on provided design
- Minimalist infantile animations using Framer Motion
- Simple, clean UI for baby shower invitation

## Common Commands

```bash
npm run dev          # Start development server
npm run build        # Build for production
npm run preview      # Preview production build
npm run lint         # Run ESLint
npm run type-check   # Run TypeScript check
```

## Architecture Overview

- **Framework**: React with Vite for fast development
- **Styling**: Tailwind CSS for utility-first styling
- **Animations**: Framer Motion for smooth, minimalist animations
- **TypeScript**: For type safety
- **Color Palette**: Light blue (celeste) theme with pastels

## Development Notes

- Focus on minimalist, child-friendly animations
- Keep UI simple and clean for invitation purposes
- Use soft, gentle transitions and micro-interactions
- Maintain accessibility for all users
